# Last Me: Survival

A 2D top-down "auto-shooter" survival game built with Pygame. Dodge hordes of zombies, pass through mathematical gates to upgrade your stats, and survive 40 waves to face the final boss.

## Features

- **Action Gameplay**: Survival-focused combat with auto-aim (toggleable).
- **Mathematical Gates**: Pass through gates that modify your damage, fire rate, and more.
- **Persistent Progression**: High scores (Best Combo) and Total Gold are saved across sessions.
- **Dynamic Waves**: Face different enemy types, blood moons, and bosses.
- **Upgrade System**: Level up to choose from randomized power-ups and stat boosts.

## Controls

- **Move**: `A` / `D` or `Arrow Keys`
- **Mouse**: Click and drag to move horizontally.
- **Pause**: `P`
- **Menu/UI**: Mouse clicks to toggle settings or select upgrades.

## Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/last-me-survival.git
   cd last-me-survival
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the game**:
   ```bash
   python main.py
   ```
